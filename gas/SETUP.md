# GAS デプロイ手順

## 1. プロジェクト作成
1. [Google Apps Script](https://script.google.com) にアクセス
2. 「新しいプロジェクト」をクリック
3. プロジェクト名を「営業アシスト」に変更

## 2. ファイル配置
1. `コード.gs` の中身を `Code.gs` の内容で置き換える
2. 左の「+」→「HTML」→ ファイル名を `Index` にする（.htmlは自動付与）
3. `Index.html` の内容を貼り付ける

## 3. デプロイ
1. 右上の「デプロイ」→「新しいデプロイ」
2. 歯車アイコン →「ウェブアプリ」を選択
3. 設定:
   - 説明: 営業アシストツール
   - 次のユーザーとして実行: **自分**
   - アクセスできるユーザー: **組織内の全員**（← ここが重要）
4. 「デプロイ」をクリック
5. 表示されたURLが社内専用アクセスURL

## 4. 共有
- デプロイURLをチームに共有するだけ
- 組織外のGoogleアカウントではアクセス不可
- Googleワークスペースへのログインが認証になる

## 注意事項
- ブラウザはChrome推奨（音声認識API対応）
- 初回アクセス時に「承認が必要です」と出たら許可する
- 更新時は「デプロイを管理」→「新しいバージョン」で再デプロイ
